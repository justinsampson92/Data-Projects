Speeches & Market Impact : An Analysis

Price Movement and Sentiment Analysis Project -- S&P 500 and VIX Indices

View project onGitHub

Justin Sampson, Brandon Flannery

Attached is a repository for our "Speeches & Market Impact: An Analysis" Project. Coming into this project we had a pressing curiosity: "Do political speeches have a direct effect on the financial markets? If so, how large or long-lived is that effect? And is there a way to quantitatively analyze a given speech, and then correlate that analysis of the speech to market performance?" In order to answer these questions we had a very methodical approach.

1) Create a program for analyzing speeches

2) Populate a data set of speeches from several financially significant political figures.

3) Populate another data set of minute by minute price data on the days those speeches were delivered.

4) Collect other types of data on the days of these speeches that would act as a control to our analysis.

5) Write more scripts in order to analyze this multitude of data vectors.

You will see in this project the culmination of all of these steps, followed by a conclusion on the findings of our analysis. Within this repository is all of the script, data, and analysis files used in the composition and execution of our analysis. Here is a description of the files in this repository for your exploratory reference:

Report Files

stat133projectreport.ipynb- This notebook contains a complete project report with reported difficulties, findings, phenomena, and a final conclusion.

Data and Functions

data- this is a folder containing links to our raw data sets as well as our cleaned data sets used in analysis.

script- this is a folder containing all of the individual scripts written in order to clean the data and allow for analysis. All of the scripts in this file are written in R. Further explanation of these individual scripts and functions can be found in the stat133datacleaning.ipynb and stat133datagathering.ipynb notebooks.

sentimentanalysis.py- This sentiment analysis file includes several python functions used in order to sort and analyze the data set of speeches. These functions are used in this analysis to 1) Extract the collected speeches into a python list, 2) Sort the speeches so their list index matches their placement in the original data file, 3) Apply a Sentiment Analysis function over this list in order to produce a sentiment value data vector that applies to the collection of speeches.

sentiments.csv- This is a dictionary of words and corresponding sentiment values used in order to calculate the sentiment of a given speech.

sanalyzeFRB.py/stat133speechextract.py- a separate function, included in sentimentanalysis.py, intended to extract speeches from their html format.

stat133proj2speeches.html- html file containing all of the speeches used in this analysis.

stat133proj2data- file containing all of the speech data in a csv format.

stat133proj2dataprice- file containing all price data in a cdv format. This price data was collected through use of a Bloomberg Terminal.

stat133proj2R.R- here is a file containing all of our R scripts for cleaning and further population the final data set
